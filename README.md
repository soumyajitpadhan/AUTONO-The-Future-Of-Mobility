# Autovate

Autovate is an app that enables cars to drive themselves. It uses smart technology and AI to help the car make decisions based on its surroundings. This helps the car navigate tricky roads, adapt to changing conditions, and keep everyone safe. Autovate‚Äôs goal is to make driving safer, smarter, and more efficient with advanced technology.

## üöÄ Getting Started

Follow these simple steps to set up the project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/soumyajitpadhan/Autovate-The-Future-Of-Mobility.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

The website should now be running at `http://localhost:5173`.
